# IMBD-capstone



## description
We created a copy of the IMDB website. Our focus for the project was to improve the design of the web page and to add new social features to  make the webpage more interesting for users to interact with via. We added a system that rewards  users with points for leaving a post and  receiving likes which is meant to make interacting with the site seem more rewarding. we also added a followers system we thought this would add more depth allowing users to find new movies by following users with similar interest

  
## Technologies
HTML/CSS,
Django,
Python,
JavaScript, 
Materilize CSS,
